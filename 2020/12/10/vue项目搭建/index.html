vue项目搭建
2020-12-10
vue-cli axios vue-router element-ui
技术

  本文章回顾了本人在学习过程中的vue-cli环境配置，axios、vue-router以及element-ui组件的安装。

vue-cli 环境配置

创建vue-cli项目之前，要安装好node.js、npm以及vue-cli脚手架，在之前的vue脚手架搭建文章中做过介绍，这里不再重复，下面从不同点开始回顾。

vue-cli 项目创建及运行

在根目录下：

创建：终端输入vue create myprogram

注意1： vue create是vue-cli3.x的初始化方式，目前模板是固定的，模板选项可自由配置，创建出来的是vue-cli3的项目，与cue-cli2项目结构不同，配置方法不同，具体配置方法参考官方文档。而vue init是vue-cli2.x的初始化方式webpack是官方推荐的标准模板名。
注意2： 这里项目名称不能含有大写字母。
回车后会显示如下界面，选择Manually select features，然后回车

空格取消Linter / Formatter，然后回车

我这里选择的是2.x版本，然后一路回车即可，创建成功后如下图所示

运行：从根目录进入项目根目录并运行，终端输入cd myprogram，npm run serve


浏览器输入http://localhost:8080/,界面如下所示

想要停止运行的话，在终端界面下按control+C即可

axios 安装

安装 axios

在项目根目录下：

安装：终端输入npm install --save axios 即可

vue-router 安装

安装vue-router

在项目根目录下：

安装：终端输入npm install vue-router

配置main.js

用vscode编译器打开该项目，其目录如下图所示

修改main.js

  • 导入vue-router

    import VueRouter from 'vue-router'

  • use 一下

    Vue.use(VueRouter)

  • 创建路由

    const router = new VueRouter({
     routes: [
         {//举个例子，若路由地址为····/test,则调用组件名为test的组件
         path:'/test', //路由地址
         component: test //组件名称

         },
     ]
})

  • 挂载到vue实例

    new Vue({
render: h => h(App),
 // 挂载到Vue示例上
 router
}).$mount('#app')

    修改好后的main.js代码如下

      import Vue from 'vue'
  import App from './App.vue'

  Vue.config.productionTip = false

  //导入路由
  import VueRouter from 'vue-router'
  //use一下
  Vue.use(VueRouter)
  // 创建路由
  const router = new VueRouter({
  routes: [
      {//举个例子，若路由地址为····/test,则调用组件名为test的组件
      path:'/test', //路由地址
      component: test //组件名称

      },
  ]
  })
  new Vue({
  render: h => h(App),
  // 挂载到Vue示例上
  router
  }).$mount('#app')

element-ui 安装

安装 element-ui

在项目根目录下：

安装：终端输入npm i element-ui

配置main.js

  • 导入element-ui

    import ElementUI from 'element-ui' // 导入 Element_ui

  • 导入 Element-ui 样式

    import 'element-ui/lib/theme-chalk/index.css'// 导入 Element-ui 样式

  • use一下

    Vue.use(ElementUI)// 插件 Element-ui

    最终main.js代码如下：

      import Vue from 'vue'
  import App from './App.vue'

  Vue.config.productionTip = false

  // 导入 Element_ui
  import ElementUI from 'element-ui'
  // 导入 Element-ui 样式
  import 'element-ui/lib/theme-chalk/index.css'
  // 插件 Element-ui
  Vue.use(ElementUI)

  //导入路由
  import VueRouter from 'vue-router'
  //use一下
  Vue.use(VueRouter)
  // 创建路由
  const router = new VueRouter({
  routes: [
      {//举个例子，若路由地址为····/test,则调用组件名为test的组件
      path:'/test', //路由地址
      component: test //组件名称

      },
  ]
  })
  new Vue({
  render: h => h(App),
  // 挂载到Vue示例上
  router
  }).$mount('#app')
