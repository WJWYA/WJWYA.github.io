position定位用法复习
2020-11-12
css positiond
技术

  我之前一直对position属性的relative值（相对定位）和absolute值（绝对定位）用法比较模糊，一直是用什么查什么，没有去深刻的理解。今天对position属性进行了系统全面学习和理解。

position定位

  position属性指定一个元素（静态的，相对的，绝对或固定）的定位方法的类型。

值 描述
static 默认值。没有定位，元素出现在正常的流中
（忽略top，bottom，left，right或者z-index声明）
fixed 生成绝对定位的元素，相对于浏览器窗口进行定位
元素的位置通过left，top，bottom以及right属性进行规定
absolute 生成绝对定位的元素，相对于static定位以外的第一个父元素进行定位
元素的位置通过left，top，bottom以及right属性进行规定
relative 生成相对定位的元素，相对于其自身正常位置进行定位
例如，”left:10px”会像元素的左侧位置添加20个像素

position:relative（相对定位）

以当前为参照物移动指定的距离,即为相对于该元素在文档流（normal flow）的原位置来定位。
注意：被定位的元素会占据原有的物理位置。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Crash Course</title>
    <style>
        #box{
            height: 300px;
            width: 600px;
            background-color: cadetblue;
        }
        #box>div:nth-child(1){
            height: 100px;
            width: 500px;
            background-color: brown;
        }
        #box>div:nth-child(2){
            height: 100px;
            width: 500px;
            background-color: darkorange;
            position: relative;/*这里添加了relation*/
        }
        #box>div:nth-child(3){
            height: 100px;
            width: 500px;
            background-color: darkgreen;
        }
    </style>
</head>
<body>
    <div id="box">
        <div></div>
        <div></div>
        <div></div>
    </div>
</body>
</html>


虽然对#box>div:nth-child(2)添加了position:relative，但没有对对其设置left，top等属性，因此它会出现在normal flow中的正常位置。
对#box>div:nth-child(2)进行修改，添加top，left属性：

  #box>div:nth-child(2){
            height: 100px;
            width: 500px;
            background-color: darkorange;
            position: relative;/*这里添加了relation*/
            left:50px;
            top: 50px;
        }


从该图片可以看出box下的第2个div元素的位置相对于其原来的位置进行了移动。

position:absolute（绝对定位）

绝对定位的元素不会占据原有的物理位置,是通过指定元素相对于最近的非static定位的父元素来确定元素位置的定位方式。
关于绝对定位的参考点：

  1. （1）如果元素的外层元素是非static（有了除默认值之外的定位设置），那么这个外层元素就成为该元素的定位参考点。
  2. （2）如果元素的外层元素没有设置任何的position的值，那么该元素将寻找距离自己最近的其他设定过position的外层元素作为参照物体（必须为嵌套层）。
  3. （3）如果该元素的外层元素没有任何一个元素采用position定位，那么此时定位参考元素变为body或者页面.
    仅修改#box>div:nth-child(2)的position属性值为absolute，不对其父元素进行修改：
 #box>div:nth-child(2){
            height: 100px;
            width: 500px;
            background-color: darkorange;
            position: absolute;/*这里修改为absolute*/
            left:50px;
            top: 50px;
        }


从图片中可以看出，box下的第2个div元素的位置相对于页面左边和顶部的距离为50px，且不在normal flow中。
在#box中添加position非static值：

 #box{
            height: 300px;
            width: 600px;
            background-color: cadetblue;
            position:relative;
        }


从图片中可以看出，box下的第2个div元素的位置相对与其父元素box的左边和顶部的距离为50px。

position:static(静态定位)

静态定位指定元素使用正常的布局行为，即元素在文档常规流中当前的布局位置。此时 top、right、bottom、left 属性无效。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>static </title>
</head>
<style>
    #box{
        height: 300px;
        width: 100%;
        background-color: cadetblue;
    }
    #content{
        height: 200px;
        width: 50px;
        background-color: cornflowerblue;
        left: 100px;/*不起作用*/
        top:50px;/*不起作用*/
        position: static;
    }
</style>

<body>
    <div id="box">
        <div id="content"></div>
    </div>
</body>
</html>


position的默认值为staitic，因此在#content中写与不写position属性，top与left等均不起作用，元素会以正常的normal flow形式呈现。

positon:fixed(固定定位)

固定定位不为元素预留空间，而是通过指定元素相对于屏幕视口（viewport）的位置来指定元素位置。元素的位置在屏幕滚动时不会改变，该元素脱离了normal flow。
修改#content中position属性值以及top和right值。

#content{
        height: 200px;
        width: 50px;
        background-color: cornflowerblue;
        right: 8px;
        top:58px;
        position: fixed;
    }


可以看出蓝色块被固定在了距离页面右侧8px，距离页面顶部58px的位置，且页面滑动时，蓝色块的位置不会发生变化。

补充：z-index堆叠顺序

  • 一旦修改了元素的定位方式，则元素可能会发生堆叠
  • 可以使用z-index属性来控制元素框出现的堆叠顺序
  • z-index仅能在定位的元素上生效
  • z-index属性：
    • 值为数值，数值越大表示对鞋顺序越高，即离用户越近
    • 可以设置为负值，表示离用户更远，一般不要设置
    • z-index仅能在定位元素上生效
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<style>
    .one{
        width: 200px;
        height: 200px;
        position:absolute;
        background-color: aquamarine;
    }
    .two{
        width: 200px;
        height: 200px;
        background-color: cadetblue;
        position: absolute;
        left: 50px;
        top: 50px;
    }
</style>
<body>
    <div class="one"></div>
    <div class="two"></div>
</body>
</html>


如图所示，two元素压在one元素上面，如想使one元素压在tow元素上面，可以在.one中添加z-index属性，仅需使其上移一层即可，当然上移n层也完全可以！

.one{
        width: 200px;
        height: 200px;
        background-color: aquamarine;
        position: absolute;
        z-index: 1;
    }

总结

  static定位是默认值；fixed定位可以固定在页面具体位置，脱离文档流；relative定位是相对于元素原始位置的位移，且不脱离文档流；absolute定位在通过最近的非static定位的父元素来进行定位，若不存在这样的父元素，就会根据页面进行定位，该定位会脱离文档流。

以上内容均为个人理解总结，若有错误或不解，可在评论区留言，亦可联系博主进行修改和讨论
