Flex布局
2020-11-13
flex css
技术

  Flex（Flexible Box）意为弹性布局，具有灵活方便的特点，且任何一个容器都可以指定为Flex布局。我参考了阮一峰老师的Flex布局教程：语法篇,对Flex的理解与实践进行了记录。

Flex布局

认识Flex

Flex可以指定任何一个容器为Flex布局，能够为盒模型提供最大的灵活性。

.box{
    display:flex;
}

即使是行内元素，也可以使用Flex布局：

.box{
    display:inline-flex;
}

考虑对浏览器的兼容性，对于webkit内核的浏览器，需要先加上-webkit前缀：

.box{
  display: -webkit-flex; /* Safari */
  display: flex;
}

当对元素使用Flex布局之后，其子元素的浮动属性float、clear以及垂直对齐vertical-align属性将会生效。

Flex的基本概念

元素采用Flex布局之后被称为Flex容器（flex container），简称“容器”。他的所有子成员自动变为容器成员，成为Flex项目（flex item），简称“项目”。下面是一个容器和项目的简单示意图，它能帮助我们认识flex布局中的一些基本概念。

容器默认有两个轴，主轴（main axis）和交叉轴（cross）。其中，默认的主轴为水平方向，交叉轴为垂直于主轴的竖直方向。主轴的开始位置（与边框的交叉点）叫做main start，结束位置叫做main end；交叉轴的开始位置叫做cross start，结束位置叫做cross end。

其中每个项目默认沿主轴排列。单个项目占据的主轴空间叫做main size，占据的交叉轴空间叫做cross size。

Flex的容器属性

Flex的容器属性有以下六个，这些属性直接设置在容器上：

  • flex-direction //设置主轴方向
  • flex-wrap //是否换行
  • flex-flow //flex-direction和flex-wrap的合并简写
  • justify-content //项目在主轴上的对齐方式
  • align-item //项目在交叉轴的对齐方式
  • align-content //多轴线对齐方式

flex-direction属性

flex-direction可以设置4个值，分别是：

  • row（默认值）：主轴为水平方向，起点在左端。
  • row-reverse：主轴为水平方向，起点在右端。
  • column：主轴为垂直方向，起点在上沿。
  • column-reverse：主轴为垂直方向，起点在下沿。
.box {
  flex-direction: row | row-reverse | column | column-reverse;
}

flex-wrap属性

在默认的情况下，项目会排在一条轴线上，当一条轴线排不下所有项目时，通过flex-wrap属性可以设置如何换行。它可以设置三个值，分别为：

  • nowrap（默认值）：不换行
  • wrap：换行，第一行在上方
  • wrap-reverse：换行，第一行在下方
.box{
  flex-wrap: nowrap | wrap | wrap-reverse;
}
flex-wrap:nowrap;

默认为不换行：

flex-wrap:wrap;

换行，第一行在上方：

flex-weap:wrap-reverse;

换行，第一行在下方:

flex-flow属性

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap。

.box {
  flex-flow: <flex-direction> || <flex-wrap>;
}

justify-content属性

justify-content属性可以设置项目在主轴上的对齐方式，包括以下六种（这里默认主轴为水平从左到右的方向）：

  • flex-start（默认值）：左对齐
  • flex-end：右对齐
  • center： 居中
  • space-between：两端对齐，项目之间的间隔都相等。
  • space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。
  • space-evenly：每个项目之间及两侧间隔均相等。//这个在阮一峰老师的原文中没有，兼容性比较差
.box {
  justify-content: flex-start | flex-end | center | space-between | space-around;
}

align-item属性

align-item属性可以设置项目在交叉轴上的对齐方式，主要包括以下五种（这里默认主轴为水平从左到右的方向，交叉轴为垂直的从上到下的方向，且justify-content值为space-around）：

  • flex-start：交叉轴的起点对齐。
  • flex-end：交叉轴的终点对齐。
  • center：交叉轴的中点对齐。
  • baseline: 项目的第一行文字的基线对齐。
  • stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

align-content属性

align-content属性可以设置多跟轴线的对齐方式，如果项目只有一根轴线，该属性不起作用。该属性可以取六个值：

  • flex-start：与交叉轴的起点对齐。
  • flex-end：与交叉轴的终点对齐。
  • center：与交叉轴的中点对齐。
  • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch（默认值）：轴线占满整个交叉轴。
.box {
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

Flex的项目属性

Flex的项目属性有以下六个，这些属性直接设置在项目上：

  • order //设置项目的排列顺序
  • flex-grow //设置项目的放大比例
  • flex-shrink //设置项目的缩小比例
  • flex-basis //设置项目占据主轴的空间
  • flex //flex-grow,flex-shrink和flex-basis的简写
  • align-self //设置单个项目的对齐方式

order属性

order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。对第四个div设置为-1,则这个div会排在第一位，因为其他div的order属性默认为0。

.box div:nth-child(4){
        order: -1;
    }

flex-grow属性

flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

设置所有项目的flex-grow的值均为1，如果存在剩余空间的话，那么它们将均分剩余的空间：

.box div{
    flex-grow:1;/*默认*/
}


若给其中一个项目的flex-grow的值设为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

 .box div:nth-child(2){
        flex-grow:2;
    }

flex-shrink属性

flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

设置所有项目的flex-shrink的值均为1，如果不存在剩余空间，所有的项目将同比缩小：

.box div{
     flex-shrink: 1;/*默认*/
}


若给其中一个项目的flex-shrink的值设为0，其他项目为1，且空间不足时，前者大小不变。

.box div:nth-child(3){
        flex-shrink: 0;
    }


注意：负值对该属性无效

flex-basis属性

flex-basis属性生效的前提是存在多余空间，该属性可以设置项目占据主轴的空间，它的默认值为auto，即项目原本的大小。除了设置auoto值外，还可以像设置width值一样来设置具体的大小（如200px），项目会占据固定的空间大小。

设置某个项目的flex-basis值为300px:

 .box div:nth-child(3){
        flex-basis: 300px;
    }

flex属性

flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto,后两个属性可以不写。阮一峰老师建议优先使用这个属性，而不是单独的写三个分离的属性。该属性有两个快捷值：auto（1 1 auto）和none（0 0 auto）。

.item {
  flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]
}

align-self属性

align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖父元素的align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。该属性可设置六个值：

  • auto
  • flex-start
  • flex-end
  • center
  • baseline
  • stretch

项目align-self属性的值除了auto外，其他属性值均与容器属性align-item的属性值相同。

总结

  flex布局阮一峰老师讲的很详细，我在这里又重新学习了一下。其中flex-grow属性和flex-shrink属性的默认值不同，flex-grow属性的默认值是0，flex-shrink属性的默认值是1，这里要怎么记忆呢。我觉得可以这么理解，要使项目大小保持原始状态，这两个属性的值就都设为0，此时flex-grow属性不会扩大填充剩余空间，flex-shrink属性在空间不足时也不会同比缩小，项目保持了原有的大小，即使超过了容器的大小。言简意赅就是当值为0的时候，项目本身多大就是多大。
